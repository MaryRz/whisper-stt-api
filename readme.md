# Whisper STT API

> A production-oriented Speech-to-Text API built with FastAPI and OpenAI Whisper.

---

## Overview

Whisper STT API is a modular backend service designed to convert speech into text using OpenAI Whisper.

The project is being developed with production-ready software engineering practices, including clean architecture, service separation, testing, containerization, and documentation.

---

## Current Features

- FastAPI backend
- OpenAI Whisper integration
- Modular project architecture
- Health check endpoint
- Swagger UI

---

## Technology Stack
